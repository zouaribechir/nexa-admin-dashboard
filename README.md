# 🚀 Nexa Admin Dashboard

A modern, premium SaaS Admin Dashboard built with HTML, CSS, and JavaScript.

Designed with a clean interface, responsive layout, smooth animations, and enterprise-level components, this project demonstrates modern dashboard design principles suitable for analytics platforms, CRM systems, and SaaS applications.

## 🌐 Live Demo

https://zouaribechir.github.io/nexa-admin-dashboard/

## ✨ Features

- 🎨 Modern & Premium UI
- 🌙 Dark / Light Mode
- 📱 Fully Responsive Design
- 📊 Interactive Analytics Dashboard
- 📈 Charts & Statistics
- 🤖 AI Assistant Interface
- 📅 Calendar
- 📋 Kanban Board
- 🔔 Notification Center
- 👥 Customer Management
- 📦 Product Management
- 🛒 Orders Management
- 📂 File Manager
- ⚙️ Settings Page
- 👤 User Profile
- 🔍 Global Search
